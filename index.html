🔥 FIREBASE SETUP

Для реальных звонков нужен бесплатный Firebase проект:

1. Зайди на console.firebase.google.com

2. Создай проект → Realtime Database → Europe → Test rules

3. В настройках проекта скопируй конфиг

4. Вставь в код в секцию firebaseConfig

5. В Rules поставь: "read": true, "write": true

В Realtime Database Rules:

{"rules":{".read":true,".write":true}}

⚠️ Test rules подходят только для частного использования. Для продакшена настрой auth.
